This AI-powered analytics dashboard is a comprehensive hackathon prototype that provides real-time data insights and interactive visualization using Llama 3 LLMs integrated via the high-speed Groq API. The implementation features a modern user interface designed for intuitive data monitoring, a robust back-end for processing complex datasets, and specialized prompt engineering to ensure contextual accuracy in AI-generated reports. The system is designed for high performance and scalability, allowing users to gain actionable insights from raw data instantly. To run the project locally, users should install the necessary dependencies from the requirements file and provide a valid Groq API key in the environment configuration as specified in the repository structure.
